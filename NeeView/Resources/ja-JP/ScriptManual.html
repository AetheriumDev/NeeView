NeeView スクリプトマニュアル

概要

スクリプトでコマンドを拡張します。複数のコマンドをまとめて実行したり、コマンドパラメーターを変更した別のコマンドとして実行することができます。

スクリプト機能はデフォルトでは無効になっています。設定の「コマンド」ページで設定して有効にします。

  • スクリプトは Javascript (ECMA 5.1) で記述します。
  • スクリプトエンジンに Jint を使用しており、.NETのクラスを使用できます。
  • ファイル拡張子は .nvjs です。
  • 文字コードはUTF8です。BOM無し推奨です。

コマンドスクリプト

スクリプトフォルダーにスクリプトファイルが配置されているとコマンドとして使用可能になります。 １ファイルが１コマンドになります。

  • 設定の「コマンド設定」で通常のコマンドと同様にショートカットを設定できます。
  • ファイル名がコマンド名になります。

イベントスクリプト

特定の名前のスクリプトファイルは自動的に実行されます。

OnBookLoaded.nvjs
ブックを開いたときに実行されます。

コンソール

スクリプトのテストをするためのコンソールです。メニューの「オプション > コンソールを開く」で開きます。

コンソール専用コマンド

cls
画面のクリア(Ctrl+L)
exit
コンソールを閉じる
help, ?
スクリプトヘルプを開く

 

スクリプトリファレンス

NeeView専用コマンドの使用方法を説明します。

include(path)

スクリプトを読み込んで実行します。

引数

path: string
スクリプトのパス。相対パスは現在のスクリプトの場所が基準になります。

戻り値

object
スクリプトの実行結果

使用例


        // Sample.nvjs を読み込んで実行
        include("Sample.nvjs")

log(message)

コンソールにメッセージを出力します。コンソールからスクリプトを実行したときのみ機能します。スクリプトテスト用です。

引数

message: string
出力メッセージ

system(filename, args)

外部アプリを実行します。

引数

filename: string
実行ファイルのパス。URLやファイルパスを指定した場合は関連付けられたアプリが実行されます。
args: string
実行ファイルの引数 (省略可)

nv.ShowMessage(message)

画面にメッセージを表示します。

引数

message: string
出力メッセージ

nv.ShowToast(message)

メッセージをトースト表示します。

引数

message: string
出力メッセージ

nv.ShowDialog(title, message, commands)

ダイアログを表示します。

引数

title: string
主文
message: string
メッセージ (省略可)
commands: int
0: OK / 1: OKCancel / 2: YesNo
ボタンの種類 (省略可)

戻り値

bool
肯定的選択("OK"もしくは"Yes")の場合はtrue、それ以外はfalseを返す。

例


        isYes = nv.ShowDialog("Is this a pen?", "I think it's a pen, really?", 2)
        if (isYes) {
            nv.ShowDialog("Good.")
        }

nv.Config

アプリの設定値を参照、変更します。 設定値の種類は後述の「設定値一覧」を参照してください。

使用例


        // テーマカラーを黒にする
        nv.Config.Theme.PanelColor = "Dark"
        nv.Config.Theme.MenuColor = "Dark"

nv.Values

値の保存のためのオブジェクトです。このオブジェクトに保存された値はアプリ起動中保持されます。

使用例


        // 値の保存
        nv.Values["Test"] = "Hello!"
        // 次の書き方でも同じ
        nv.Values.Test = "Hello!"

nv.Book

ブック情報へのアクセサを取得

戻り値

BookAccessor
ブックアクセサ

[Type] BookAccessor

ブックアクセサ

プロパティ

Path: string
現在のブックのパス。ブックが開かれていない場合はnull
IsMedua: bool
現在のブックが動画であるかどうか
IsNew: bool
現在のブックが履歴のない新しく開かれたブックであるか
PageSize: int
現在のブックのページ数
ViewPageSize: int
現在表示されているページ数
Config.ViewPageSize: int
表示するページ数
Config.BookReadOrder: string
左開き/右開き [ LeftToRight, RightToLeft ]
Config.IsSupportedDividePage: bool
横長ページを分割する
Config.IsSupportedSingleFirstPage: bool
最初のページを単独表示
Config.IsSupportedSingleLastPage: bool
最後のページを単独表示
Config.IsSupportedWidePage: bool
横長ページを2ページとみなす
Config.IsRecursiveFolder: bool
サブフォルダーを読み込む
Config.SortMode: string
並び替え順 [ FileName, FileNameDescending, ...]

BookAccessor.Page(number)

ページ取得

引数

page: int
ページ番号(1-n)。表示に合わせているため、1スタートである点に注意してください。

戻り値

PageAccessor
ページアクセサ。取得できなかった場合はnullを返す。

BookAccessor.ViewPage(index)

表示ページ取得

引数

index: int
表示ページID(0-1)

戻り値

PageAccessor
ページアクセサ。取得できなかった場合はnullを返す。

[Type] PageAccessor

ページアクセサ。ページ情報をプロパティとして保持しています。

プロパティ

Path: string
ページのパス
Width: double
ページの幅
Height: double
ページの高さ

使用例


        // 表示ページ名をトースト表示
        nv.ShowToast(nv.Book.ViewPage(0).Path)

nv.Command(commandName)

コマンドのアクセサを取得します。 コマンドは 設定 > コマンド設定 で表示されるコマンドと同じものです。詳細は後述する「コマンド一覧」を参照してください。

引数

commandName: string
コマンド名

戻り値

CommandAccessor
コマンドアクセサ。コマンドが見つからない場合はnull。

[Type] CommandAccessor

コマンドアクセサ

CommandAccessor.Parameter: json

コマンドパラメーターの取得、設定ができます。設定は恒久的なものです。

使用例


        // ViewRotateLeft コマンドの回転量を45度に変更
        nv.Command("ViewRotateLeft").Parameter.Angle = 45

CommandAccessor.ShortCutKey: string

コマンドのショートカットの取得、設定ができます。

使用例


        // ViewRotateLeft のショートカットに Ctrl+A を設定
        nv.Command("ViewRotateLeft").ShourtCutKey = "Ctrl+A"

CommandAccessor.MouseGesture: string

マウスジェスチャーの取得、設定ができます。

ジェスチャーキーは次のようにアルファベットで指定します。

  • ← ... L
  • ↑ ... U
  • → ... R
  • ↓ ... D
  • Click ... C

使用例


        // ViewRotateLeft のマウスジェスチャーに ↑←↓ を設定
        nv.Command("ViewRotateLeft").MouseGesture = "ULD"

CommandAccessor.TouchGesture: string

タッチ操作の取得、設定ができます。

使用例


        // ViewRotateLeft のタッチ操作に TouchCenter を設定
        nv.Command("ViewRotateLeft").TouchGesture = "TouchCenter"

CommandAccessor.Execute(args...)

コマンドを実行します。

引数を指定可能なコマンドがあります。 これらは通常はダイアログが表示されるコマンドで引数を指定することでユーザーの選択を省略するといった用途に使用できます。

引数

args: object[]
コマンド引数。コマンドによって内容は異なります。可変長引数です。(省略可)

戻り値

bool
コマンドが発行できたか。ロード中などにコマンドが発行できないことがある。コマンド自体の成否は判別しない。

使用例


        // C:\\Foo\\Bar.zip を開く
        nv.Command("LoadAs").Execute("C:\\Foo\\Bar.zip")

CommandAccessor.Patch(patch)

一時的にコマンドパラーメーターを変更します。

引数

patch: json
一時的に上書きするコマンドパラメーターをJSON形式で指定します。

戻り値

CommandAccessor
自身のコマンドアクセサをそのまま返す

使用例


        // 5度指定で左回転を実行
        nv.Command("ViewRotateLeft").Patch({"Angle": 5}).Execute()
